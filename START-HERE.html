⚠️ Important: How to Open HTML Files

If you see "code" when opening files, you're viewing the SOURCE CODE. Follow the steps below to open them correctly!

📥 Step 1: Download All Files

What You Need:

Download ALL these files to a folder on your computer:

  • admin-login.html - Admin login page
  • admin-dashboard.html - Admin control panel
  • landing-page.html - Your homepage
  • catalog.html - Fruit catalog
  • fruit-detail.html - Product details
  • build-your-box.html - Shopping cart

💻 Step 2: Open Files Correctly

On Windows:
  1. Right-click the .html file
  2. Select "Open with" → "Google Chrome" (or your browser)
  3. ✅ The page should render, not show code!
On Mac:
  1. Right-click the .html file
  2. Select "Open With" → "Safari" (or Chrome)
  3. ✅ The page should render, not show code!
Alternative Method (Any System):
  1. Open your web browser (Chrome, Safari, Firefox)
  2. Press Ctrl+O (Windows) or Cmd+O (Mac)
  3. Navigate to the folder with HTML files
  4. Select the file and click "Open"
  5. ✅ Page should render correctly!

🔐 Step 3: Login to Admin

Access Your Admin Dashboard:
  1. Open admin-login.html in your browser
  2. Username: admin
  3. Password: dreamfarm2024
  4. Click "Login to Admin Dashboard"
  5. ✅ You're now in your admin panel!

✅ Step 4: Test Everything

Quick Test Checklist:
  • ✅ Open landing-page.html - See your homepage
  • ✅ Click "Browse Fruits" - See the catalog
  • ✅ Click any fruit - See details and health benefits

🐛 Still Seeing Code?

If files still open as text/code:

  1. Make sure files end with .html (not .txt)
  2. Try a different browser (Chrome recommended)
  3. Don't open from Downloads folder - move to Desktop first
  4. Right-click → Properties → "Open with" → Set default to Browser

📱 Mobile Access

To Edit From Your Phone:
  1. Upload all files to a web server (or use a local server)
  2. Open admin URL in phone browser
  3. Login with credentials above
  4. ✅ Edit products from anywhere!

Note: Files must be on a web server to access from phone. They won't work if just saved on your computer.
